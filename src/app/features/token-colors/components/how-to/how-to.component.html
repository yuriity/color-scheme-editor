There is a few options to fix this

  • Open the existing color theme file: File -> Open local file...
  • Import color scheme from copy-pasted contents of the color theme file: File -> Import from copied JSON...
  • Generate your color scheme from VSCode directly
    1. Open VSCode
    2. Press (CMD or CTRL) + SHIFT + P
    3. Enter: > Developer: Generate Color Scheme From Current Settings and copy the contents
    4. Open dialog: File -> Import from copied JSON... and paste
